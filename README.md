# BifferTraders
An attempt at a Rails plugin for low-level SpaceTraders functionality

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem "biffer_traders"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install biffer_traders
```
